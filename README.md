## 🚀 What I'm Building
 
### **RIVA** - Transforming Real Estate in Moldova 🏠
Currently building a platform set to digitize and modernize the real estate market in Moldova — a market that's largely offline, fragmented, and overdue for change.
 
- 📱 React Native mobile app (iOS & Android)
- 🌐 React web dashboard
- ⚙️ Node.js + PostgreSQL backend
- 🔐 Google & Apple sign-in, full design system
 
**The Challenge:** Finding or listing a property in Moldova still means phone calls, paper contracts, and outdated listings. RIVA changes that.
